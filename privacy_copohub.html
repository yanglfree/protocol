Privacy Policy

Effective Date: August 6, 2025

This privacy policy explains how we handle your data when you use our GitHub client app ("the App"). We value your privacy and take your data protection seriously.

1. Data We Do Not Collect

We do not collect, store, or share any personal data from users. The App does not use any analytics SDKs, advertising networks, or third-party trackers.

2. GitHub Account Access

The App uses the official GitHub OAuth login to authenticate you directly with GitHub. Your GitHub access token is securely stored on your device and is never sent to any external server we control. We do not have access to your password or token.

3. How the App Uses GitHub API

The App connects directly to the official GitHub API to access your repositories, issues, pull requests, contribution data, and other public or authorized content. This is done solely to provide the features of the App and is not used for any other purpose.

4. Local Data Storage

All data, including themes, settings, and widget preferences, is stored locally on your device. No data is transmitted to any third-party servers.

5. Third-Party Services

The App uses only the official GitHub API (https://api.github.com). No other third-party services are integrated.

6. Security

We follow best practices to ensure secure communication with GitHub via HTTPS. Your access token is stored securely using system-provided encryption mechanisms when possible.
